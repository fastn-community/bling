A "Read More" link is a user interface element commonly used in web content to provide a way for users to access additional or extended content beyond an initial excerpt or summary. It is typically used when the content is truncated to a certain length to save space or provide a preview, and users can click the "Read More" link to expand or view the full content.

How to use

Add bling dependency into your FASTN.ftd
-- fastn.dependency: fastn-community.github.io/bling
Lang:
ftd
import read-more inside your .ftd file
-- import: fastn-community.github.io/bling/read-more
Lang:
ftd
sample usage of -- read-more component
-- read-more.read-more: Read more: 
link: read-more/

Default position is `right`

-- read-more.read-more: Read more: 
link: read-more/
position: center

Center position of `read-more`

-- read-more.read-more: Read more: 
link: read-more/
position: left

Left position of `read-more`
Lang:
ftd

Properties:

Below are details of all properties of read-more component.
title
type:
Caption (required)
It's used to add title which is clickable.
body
type:
Body (optional)
It's used to add body content to read-more component.
link
type:
String (required)
It's used to add link to read-more component.
position
type:
String (optional, default: right)
It's used to move read-more component position. You can use start, center or right to change its position respectively.
