A "note component" typically refers to a UI component used to display a note or a small piece of information within a user interface. It is commonly used in applications, websites, or documentation systems to present brief explanations, tips, warnings, or other important information to users.

How to use

Add bling dependency into your FASTN.ftd
-- fastn.dependency: fastn-community.github.io/bling
Lang:
ftd
import note inside your .ftd file
-- import: fastn-community.github.io/bling/note
Lang:
ftd
