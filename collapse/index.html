The collapse component typically consists of a header and a collapsible body. The header serves as a clickable element, such as a button or a title, that triggers the collapse functionality.

Add bling dependency into your FASTN.ftd

Properties:

Below are details of all properties of collapse component.
title
type:
Caption (required)
It's used to add title which is clickable.
info-right
type:
String (optional)
It's used to show some info/text to the right corner of the title of collapse.
$is-open
type:
Boolean (optional, default: false)
It's used to show body content of collapse, default value is false if you set it to true then body content will be visible.
remove-gap
type:
Boolean (optional, default: false)
It's used to remove bottom margin between two collapse components, default value is false if you set it to true then bottom margin of collapse will be removed.
body
type:
Body (optional)
It's used to add body content to collapse component.
